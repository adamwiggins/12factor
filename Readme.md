The Twelve-Factor App
=====================

Source for the content app running at: http://www.12factor.net/

Development
-----------

    bundle install
    foreman start
    open http://localhost:5000

Production deploy
-----------------

    heroku create -s cedar
    git push heroku master
    heroku open
